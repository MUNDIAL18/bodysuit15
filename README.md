## Ejecutar el bot estable con PM2

1.- instalar PM2 en termux :
```bash

> npm install -g pm2
```  

2.- Ejecutar el bot en PM2 :
```bash 

> pm2 start nkbot.js
```
3.- Visualizar proceso en PM2 :
```bash 

> pm2 monit
[ ! ] Para salir del monitor usa la combinación de CTRL + c
```

4.- Eliminar PM2 para el bot :
```bash

> pm2 kill
```
